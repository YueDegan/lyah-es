¡Aprende Haskell por el bien de todos!
======================================

Traducción al español de la guía del lenguaje de programación Haskell para
principiantes [Learn you a Haskell for great good!](http://learnyouahaskell.com/).

Puedes ver la guía en [http://aprendehaskell.es](http://aprendehaskell.es) (Original)

Puedes ver la guía optimizada en [https://yuedegan.github.io/AprendeHaskell/](https://yuedegan.github.io/AprendeHaskell/) (Optimizada)

# Sobre este proyecto
Este repositorio es un proyecto de recuperación, mantenimiento y actualización de la guía ¡Aprende Haskell por el bien de todos!, una adaptación al español de Learn You a Haskell for Great Good.

## Historia del proyecto original
El sitio web AprendeHaskell.es fue creado por `Alberto Vilches (alvivi)` como una adaptación al español de la popular guía Learn You a Haskell for Great Good. El objetivo era ofrecer a la comunidad hispanohablante una introducción accesible al lenguaje de programación Haskell.

Actualmente el sitio continúa disponible en [AprendeHaskell.es](aprendehaskell.es) y mantiene la estructura clásica de la obra original, organizada en capítulos que cubren desde conceptos básicos hasta temas avanzados como funtores, mónadas y zippers.

## Tecnologías y estructura del proyecto original
El proyecto original fue desarrollado como un sitio web estático basado principalmente en:

* HTML
* CSS
* Imágenes y recursos estáticos
* Documentación generada para navegación por capítulos

La estructura del repositorio original contiene contenido organizado por capítulos, hojas de estilo, imágenes y archivos necesarios para la generación y publicación del sitio web.

## Objetivos de este repositorio
Este proyecto no pretende reemplazar la obra original ni atribuirse su autoría.

Su finalidad es:

* Preservar el contenido ante posibles pérdidas o abandono del proyecto.
* Corregir errores ortográficos, tipográficos y de formato.
* Mejorar la compatibilidad con navegadores modernos.
* Adaptar el diseño para dispositivos móviles y pantallas actuales.
* Actualizar dependencias, recursos y herramientas de construcción.
* Facilitar futuras contribuciones de la comunidad.
* Mantener disponible una referencia de aprendizaje de Haskell en español.

## Estado del proyecto
Este repositorio se considera una bifurcación de mantenimiento (maintenance fork), centrada en la conservación y modernización de una obra educativa ampliamente utilizada por la comunidad hispanohablante de programación funcional.

**Todo el mérito por la obra original, la traducción y el contenido educativo corresponde a sus autores originales.**
